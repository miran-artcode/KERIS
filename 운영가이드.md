# 🎤 강의자 운영 가이드 — 집합 ⑪·⑫ (BEYOND THE SCORE)

사이트: **https://keris-data.web.app**

## 연수 전 (5분)

1. `https://keris-data.web.app` 접속 → 푸터의 **Console** 클릭 → 코드 **1112** → 별명(예: 미란쌤) + PIN 4자리 입장
2. 강의자 콘솔에서 덱 선택 → **🔴 LIVE 시작** → 참가자 화면(home)과 동기화 확인
   - 덱 4개(진행 순서대로): **① ⑨복습(50장) → ② ⑪ 데이터 강의(40장) → ③ ⑫ 환류(33장) → ④ 로그 브리핑(11장)**
   - ⚠️ 설계안 양식 덱은 타인 작성 예시가 포함되어 있어 사이트에서 제거함 — 설계안은 각자 배부받은 종이/파일로, 칸 구조 안내는 웜업 페이지 텍스트로 제공
   - 슬라이드 아래에 **강의자 노트**(진행 멘트·스몰토크·과제 확인)가 구간별로 자동 표시됨. 전체 노트는 「📓 노트·자료」 탭
3. 프로젝터에는 `home.html`(참가자 화면)을 띄우고, 노트북에는 `presenter.html`(콘솔)을 띄우면 이중 화면 운영 가능
4. 참가자 안내 문구: **"keris-data.web.app 접속 → 교과 누르고 → 별명 + 숫자 4자리"**

## 진행 순서 (JSX 연수앱의 운영 노트 계승)

| 단계 | 시간 | 콘솔에서 할 일 |
|---|---|---|
| ① 입장·웜업 | 15′ | `poll-subject`(교과 분포)·`p3-weakest`(자신 없는 칸) 발사. 복습 카드 6장 + **설계안×이론 체크 12항목**(웜업 페이지) 진행 — 「진행·과제」 탭에서 완료 인원 확인 |
| ② 설계안 점검 | 40′ | **실시간 모니터** 탭 상시 확인 — 🔴 막힘이 위로 정렬됨. 필요시 ⑨ 복습 덱·설계안 덱을 띄워 이론 재확인 (구간별 강의자 노트 참고) |
| ③ 스피드 퀴즈 | 10′ | **리더보드** 탭을 화면에 띄워 짧게 시상 (⑪ 시작 전 환기) |
| ④ ⑪ 강의 | 100′ | **⑪ 데이터 강의 덱(40장)** LIVE — 구간별 노트에 스몰토크·발사할 퀴즈가 적혀 있음(`p4-tmap`·`s4-avg`·`s4-gigo` 등). 마지막에 **"내 데이터 열어 보기"** + s11 과제(해석 한 줄) 확인 |
| ⑤ ⑫ 강의·실습 | 100′ | ⑫ 환류 덱 LIVE → s12 갤러리 워크 → **연수실 전체 데이터** 함께 열기 → 액션 플랜 → KPT. `poll-role`, `poll-minimal` 발사. 「진행·과제」 탭에서 ⑫ 과제 3종 확인 |
| (선택) 로그 브리핑 | 10′ | ⑪ 도입부나 ⑫ 시작에 **로그 브리핑 덱(11장)** — "지금 우리가 쓰는 이 시스템"의 원리와 동의 설계 설명 |

- ⑪ 로그 시연 팁: **특정인을 지목하지 말고 익명 패턴 중심**으로 읽기. "체류가 긴 단계 = 어려움일까 몰입일까"를 발문으로.
- 발화 전사: 콘솔 **🎙️ 발화 전사** 탭 (Chrome, 마이크 허용). ⑫에서 "오늘 나의 발화"를 돌아보는 재료.
- 슬라이드 넘기기: 콘솔에서 키보드 **← →**. 추천 퀴즈가 있는 슬라이드에선 카드가 자동으로 뜸.

## 신호 체계 (참가자 화면 우하단)

🟢 따라가고 있어요 · 🟡 조금 헷갈려요 · 🔴 막혔어요(즉시 확인!) · 😂 빵 터짐(농담 적중 확인용)
→ 콘솔 **실시간 모니터**에 집계·정렬, **신호 피드**에 시간순 기록. 이 신호가 곧 멀티모달 데이터.

## 연수 후

1. 콘솔 **💾 내보내기** — JSON(전체) + CSV(신호·체류) + KPT·플랜 텍스트 저장
2. [Firebase 콘솔](https://console.firebase.google.com/project/keris-data/firestore)에서 컬렉션 삭제 (사이트에 "연수 종료 후 삭제" 안내가 있으므로 꼭 이행)
3. 다음 강의에 재사용: 저장소 `README.md`의 구조 참고 — 내용만 갈아끼우면 됨

## 만약의 상황

- **Wi-Fi 불안정**: 사이트는 오프라인이어도 입장·열람 가능(로그만 유실). Plan B: 슬라이드는 원본 PPTX로.
- **참가자가 비번을 잊음**: 별명을 살짝 바꿔 재입장 (예: 소나무 → 소나무2)
- **명단에 이상한 참가자**: 콘솔 모니터에서 행 클릭 → 숨김
